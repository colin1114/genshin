## 题目描述

给出一个有 $n$ 个节点的树 (保证 $1$ 号节点为根节点) 和 $q$ 个询问 $x$,$y$，回答 $x, y$ 的最近公共祖先。

## 输入格式

第 $1$ 行：$2$ 个整数 $n$ 和 $q$

接下来 $n-1$ 行，每行 $2$ 个整数，表示树的一条边

接下来 $q$ 行，每行 $2$ 个整数，表示一次询问

## 输出格式

共 $q$ 行，每行回答一个询问，即 $x, y$ 的最近公共祖先

## 数据范围

tarjan LCA：$1 \leq n \leq 5 \times 10^6$,$1\leq q \leq 3000$.

LCA：$1\leq n,q \leq 2 \times 10^5$.
