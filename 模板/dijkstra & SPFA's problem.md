### 问题描述

有 $N$ 座建筑物以等间隔排成一排。第 $i$ 座建筑物的高度为 $H_i$。

你希望用灯光装饰其中的一些建筑物，使得满足以下两个条件：

- 被选中的建筑物高度相同。
- 被选中的建筑物等间隔排列。

请问最多可以选择多少座建筑物？如果你只选择一座建筑物，也被认为满足上述条件。

### 条件

- $1 \leq N \leq 3000$
- $1 \leq H_i \leq 3000$
- 所有输入值均为整数。

### 输入格式

输入从标准输入给出，格式如下：

```
$N$
$H_1$ $\ldots$ $H_N$
```

### 输出格式

输出答案。
